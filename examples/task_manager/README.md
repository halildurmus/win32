A simple task manager app that allows users to manage running Windows processes.

## Features

- View a list of running tasks, including their names, PIDs, and descriptions.
- Search and sort tasks based on their name, PID, or description.
- Start a new task by specifying its executable path directly within the app.
- Terminate a running task by clicking the button next to the task.
